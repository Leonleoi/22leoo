# 22i! — past demos droppin season ;) 🌃

A personal music portfolio site for **22i!**, a Chinese indie music producer & vocalist creating lo-fi, emo-rap and alternative sounds.

🎧 Hosted on NetEase Cloud Music  
🖤 Built with a raw editorial / brutalist-inspired visual style

---

## 🌐 Live Site

👉 https://leonleoi.github.io/22leoo/

## ✨ Features

- 🌃 Dynamic LA night skyline background slideshow
- 🌐 English / 中文 language toggle
- 💿 Discography showcase
  - 1 Compilation
  - 1 EP
  - 1 Single
  - 13 tracks total
- 🎵 Full song catalog with direct NetEase Cloud Music links
- 🔮 Floating orb particle system
- 🖱️ Custom cursor effects
- 📱 Fully responsive on desktop & mobile
- 📰 Brutalist / editorial-inspired layout
