# 22i!

**22i!** — past demos droppin season ;)

A personal music portfolio site for **22i!**, a Chinese indie music producer and vocalist creating lo-fi, emo-rap &amp; alternative sounds. Hosted on [NetEase Cloud Music](https://music.163.com/artist?id=94514287).

Built as a single-page site with an editorial/brutalist aesthetic, featuring:

- Dynamic LA night skyline background with rotating photos
- EN/CN language toggle
- Discography showcase (1 compilation, 1 EP, 1 single / 13 tracks total)
- Full song catalog with direct links to NetEase Cloud Music
- Floating orb particle system &amp; custom cursor effects
- Fully responsive across desktop and mobile

Tech: vanilla HTML/CSS/JS, no frameworks. Deployed via GitHub Pages.

Live site: https://leonleoi.github.io/22leoo/

**Tools** 
This project was completed using Claude Code + Codex.

技术栈：原生 HTML/CSS/JS，无框架。通过 GitHub Pages 部署。
